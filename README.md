## What This Does

Fortnite Baby Clicker Script is a web-focused utility built for browser gameplay. Its purpose is to automate repeating click actions so you do not have to keep tapping manually during extended sessions. The script is intended for the Fortnite Baby Clicker experience and targets HTML-based, browser-hosted play.
